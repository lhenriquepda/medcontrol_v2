## Bugs novos descobertos (NÃO fechados nesta release)

### BUG-021 (#088) — Dose não aparece em Início sem refresh manual [P1]

Reportado em emulador Pixel 7 API 35. Após cadastrar dose nova, voltar pra Início mostra lista antiga — user precisa pull-to-refresh ou sair/voltar de tab. **NÃO repro em Samsung S25 Ultra device físico real.**

Provável causa: TanStack Query `invalidateQueries(['doses'])` não chamado após mutation INSERT, ou hook `useDoses` não escuta evento realtime correto, ou race condition timing emulador-only.

⚠️ Fix DEVE preservar comportamento em S25 Ultra. Test cross-device obrigatório.

### BUG-022 (#089) — Layout AdSense banner sobrepondo header [P2]

Visível emulador Pixel 7 (1080×2400 @420dpi). Print confirma: banner "Test Ad 468x60" empurra header "Dosy ▸ Frederico" parcial; tabs/cards renderizam OK abaixo. **NÃO repro em Samsung S25 Ultra.**

Provável causa: posicionamento absoluto AdSense em `index.html` ou CSS colidindo + header sem ellipsis truncate.
