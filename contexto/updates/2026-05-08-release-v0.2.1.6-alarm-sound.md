## Items fechados (1)

### #203 — Som de alarme customizado

**Implementação:**

1. **Otimização do mp3:**
   - Original: 69s, 192kbps stereo, 1.66 MB
   - Otimizado: 96kbps mono 44.1kHz, 811 KB (50% redução)
   - Tool: `ffmpeg -ac 1 -ar 44100 -b:a 96k`

2. **Recurso Android:** `android/app/src/main/res/raw/dosy_alarm.mp3`

3. **`AlarmService.java`** já tinha fallback raw (linha 255-263) — sem mudança necessária:
   ```java
   int rawId = getResources().getIdentifier("dosy_alarm", "raw", getPackageName());
   if (rawId != 0) {
       Uri uri = Uri.parse("android.resource://" + getPackageName() + "/" + rawId);
       activePlayer.setDataSource(this, uri);
   }
   ```

4. **`AlarmReceiver.java`** atualizado pra usar mesmo raw em channel sound:
   ```java
   int rawId = context.getResources().getIdentifier("dosy_alarm", "raw", context.getPackageName());
   if (rawId != 0) {
       sound = Uri.parse("android.resource://" + context.getPackageName() + "/" + rawId);
   }
   if (sound == null) sound = RingtoneManager.getDefaultUri(RingtoneManager.TYPE_ALARM);
   ```

5. **CHANNEL_ID bumped** `doses_critical` → `doses_critical_v2`. NotificationChannel sound é immutable após criação. Devices que tinham canal antigo (com som default) recebem novo canal `_v2` no upgrade. Antigo fica órfão (limpável Settings → App).
