# Release v0.2.0.3 — Header alertas + perf + dark contrast

**Data:** 2026-05-04
**Branch:** `release/v0.2.0.3`
**Versão:** 0.2.0.3 (versionCode 36)

## Items fechados

- **#116** [P1 UX] Header alertas: sino dropdown → ícones diretos.
  - Componente novo `HeaderAlertIcon` (`src/components/dosy/HeaderAlertIcon.jsx`) — primitive reusable com 4 tones (danger/warning/info/update), badge contador, anim pulse opcional pra urgência.
  - `AppHeader.jsx` reescrito: substitui `<BellAlerts alerts={...}/>` por 4 ícones condicionais. Cada ícone só renderiza se count > 0.
  - Tipos:
    - `AlertCircle` (danger, pulse) — doses atrasadas → `/?filter=overdue`
    - `Users` (info) — paciente compartilhado novo → `/pacientes` (#117)
    - `Pill` (warning) — tratamento acabando ≤3d → `/pacientes` (#118)
    - `Download` (update) — app update disponível → `startUpdate()`
  - UpdateBanner verde no topo MANTIDO (user pediu redundância intencional).
  - `BellAlerts` component fica deprecated mas exportado pra compat (zero break).
  - Animation `dosy-pulse` adicionada em `dosy-tokens.css`.

- **#117** [P2 UX] Alerta paciente compartilhado comigo.
  - `sharesService.listReceivedShares()`: query `patient_shares WHERE sharedWithUserId = auth.uid()` ordenada por `createdAt desc`.
  - `useReceivedShares()` hook (staleTime 60s).
  - Header conta shares cujo `createdAt > localStorage[dosy_shares_seen_at]`.
  - Click → `seenAt = now` (decay automático) → `nav('/pacientes')`.

- **#118** [P2 UX] Alerta tratamento acabando ≤3 dias.
  - Computa `endDate = startDate + durationDays * 86400000ms` em memória (sem coluna nova).
  - Filtra: `!isContinuous && status='active' && endDate ∈ [now, now+3d]`.
  - Decay via `localStorage[dosy_ending_seen_at]` semelhante ao #117.
  - Click → `nav('/pacientes')`.

- **#033** [Auditoria P2] React.memo em DoseCard.
  - DoseCard agora `memo()` exportado. Props shallow equal previne re-render quando dose obj não mudou (React Query cache stable).
  - PatientCard já tinha memo anterior. TreatmentCard não existe (achado falso da auditoria — só DoseCard + PatientCard listas no app).

- **#040** [Plan P2] Contraste textos secundários no dark mode.
  - `--dosy-fg-secondary` #C8B8AB → #DDC8B6 (ratio 8.7 → 10.5)
  - `--dosy-fg-tertiary` #8E7F73 → #B0A091 (ratio 4.35 → 5.8 — passa WCAG AA 4.5)
  - `--dosy-fg-disabled` #5C5048 → #6E6256 (mais visível mas ainda obviamente disabled)
  - `--dosy-border` 0.08 → 0.10 alpha
  - `--dosy-border-strong` 0.16 → 0.20 alpha (cards/dividers respiram melhor)
